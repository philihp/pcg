A functional implementation of the [PCG family random number generators](), written in JavaScript.

[PCG family random number generators]: http://pcg-random.org

## Getting started

To achieve frictionless reproducibility of random output results, immutable objects are used throughout the project.

First, seed a PCG state. A stream ID specifies _which_ unique periodic series of entropy to use. The state specifies _where_ in that series we start.

```
import { createPcg32 } from 'pcg'

const advancedOptions = {}
const initState = 42
const initStreamId = 54
const state0 = createPcg32(advancedOptions, initState, initStreamId)
```

After that, random outputs can be generated by calling appropriate functions as shown below:

```js
import { randomInt, randomList } from 'pcg'

const randomUint32 = randomInt(0, 2 ** 32 - 1)
const [value, state1] = randomUint32(state0)

const listLength = 3
const listItemRng = randomUint32
const [[v1, s1], [v2, s2], [v3, s3]] = randomList(listLength, listItemRng, state0)
```

In this above example, `value === v1` and `state1 === s1`.

## Stream schemes

`createPcg32` accepts a `streamScheme` option that selects how the LCG increment is derived from the seed. The default is `SETSEQ`.

- `SETSEQ` (default) — distinct stream per `streamId` (the standard "set-seq" PCG variant).
- `ONESEQ` — a single fixed stream; `streamId` is ignored.
- `MCG` — multiplicative congruential generator (increment = 0); shorter period, slightly cheaper per step.

```js
import { createPcg32, StreamScheme } from 'pcg'

const state0 = createPcg32({ streamScheme: StreamScheme.ONESEQ }, 42, 54)
```
